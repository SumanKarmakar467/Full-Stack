                                             index.js

                                            (APP) App.js 
                                                 ⬇️
                    Navbar.js  , WorkoutDetails.js        Home.js
                        (components)                      (pages)

## Backend 
    server.js - gets the post number in .env file useing dotenv
    ⬇️
    Routes (workout.js) - for the api calls 
    ⬇️
    Controllers(workoutController.js) - how the api calls work 

    models (workoutModel.js) - for mongoDB using mongoose to creae schema and store the data into this
